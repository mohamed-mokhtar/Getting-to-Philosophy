# Getting-to-Philosophy
### Getting Started
**Python3 is required**
These instructions will get you a copy of the project up and running on your local machine for develoment.  
> git clone https://github.com/mohamed-mokhtar/Getting-to-Philosophy  
> cd Getting-to-Philosophy  
> pip install requirements.txt  

#### Requirements
What things you need to install ?
- time
- requests
- re
- argparse
- BeautifulSoup
- urllib



### How to use?
**Open terminal/cmd in the project directory**  
python Crawl.py your_first_anchor_right_here  
Example : python Crawl.py https://en.wikipedia.org/wiki/Philosophy_of_science  

### Usecases
- Entry anchor is getting into a loop so Philosophy is not reachable.
- Entry anchor needs more than the limit specified inside the script to reach the target.
- Entry anchor can reach it correctly.
